# Student Management System v1.1

## Project Description

This is a menu-driven Student Management System developed in Python. It allows users to manage student records through a simple console interface. The project stores data in a CSV file so that student information is available even after the program is closed and reopened.

---

## Features

* Add Student
* Prevent Duplicate Student Names
* View All Students
* Search Student by Name
* Update Student Details
* Delete Student
* Count Total Students
* Save Data to CSV
* Load Data from CSV Automatically
* Input Validation for Menu Choices

---

## Technologies Used

* Python 3
* CSV Module

## How to Run

1. Install Python 3.
2. Open a terminal or command prompt.
3. Navigate to the project folder.
4. Run the program using:

python mini project(Student Management System).py
