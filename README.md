# 🎓 Student Management System

A Python-based **Student Management System** developed as part of my learning journey in software development. This repository contains two implementations of the same project, demonstrating the progression from **CSV file handling** to a **MySQL database with Object-Oriented Programming (OOP)**.

---

## 📂 Repository Structure

```text
student-management-system/
│
├── csv-version/
│   ├── Student Management System.py
│   ├── save_students.csv
│   └── README.md
│
├── mysql-version/
│   ├── student_management_oop.py
│   └── README.md
│
└── README.md
```

## 📁 CSV Version

The first version of the project stores student records using **CSV file handling**.

### Features

- ➕ Add Student
- 📋 View Students
- 🔍 Search Student
- ✏️ Update Student Details
- ❌ Delete Student
- 🔢 Count Students
- 💾 CSV File Storage
- ✅ Input Validation

### Technologies Used

- Python
- CSV File Handling

---

## 🗄️ MySQL + OOP Version

The second version improves the project by storing student records in a **MySQL database** and organizing the code using **Object-Oriented Programming (OOP)**.

### Features

- ➕ Add Student
- 📋 View Students
- 🔍 Search Student
- ✏️ Update Student Details
- ❌ Delete Student
- 🗄️ MySQL Database Integration
- 🏗️ Object-Oriented Programming (Classes & Methods)
- ✅ Exception Handling
- ✅ Input Validation
- 🔒 Parameterized SQL Queries

### Technologies Used

- Python
- MySQL
- mysql-connector-python
- Object-Oriented Programming (OOP)

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/DGugan/student-management-system.git
```

Navigate to the project folder:

```bash
cd student-management-system
```

Choose the implementation you want to explore:

- 📁 `csv-version/` – CSV File Handling
- 📁 `mysql-version/` – MySQL + OOP

Each folder contains its own README with setup instructions.
