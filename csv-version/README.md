**# 🎓 Student Management System (CSV Version)**



**A menu-driven Student Management System built using \*\*Python\*\* and \*\*CSV file handling\*\*. This project was developed to practice Python programming, file handling, and CRUD (Create, Read, Update, Delete) operations.**



**---**



**## ✨ Features**



**- ➕ Add Student**

**- 📋 View All Students**

**- 🔍 Search Student**

**- ✏️ Update Student Details**

**- ❌ Delete Student**

**- 🔢 Count Total Students**

**- 💾 Store data using CSV files**

**- ✅ Input validation and user-friendly menu**



**---**



**## 🛠️ Technologies Used**



**- Python**

**- CSV File Handling**

**## ▶️ How to Run**



**1. Make sure Python is installed.**

**2. Clone the repository:**



**```bash**

**git clone https://github.com/DGugan/student-management-system-csv.git**

**```**



**3. Navigate to the project folder:**



**```bash**

**cd student-management-system-csv**

**```**



**4. Run the application:**



**```bash**

**python "Student Management System.py"**

**```**
